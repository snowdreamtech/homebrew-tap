# snowdreamtech/homebrew-tap

[Homebrew](https://brew.sh/) tap for [snowdreamtech](https://github.com/snowdreamtech) maintained formulae and casks.

> 🇨🇳 [中文文档](README.zh-CN.md)

## Available Casks

| Cask                                              | Description                                                                           |
| ------------------------------------------------- | ------------------------------------------------------------------------------------- |
| [unirtm](https://github.com/snowdreamtech/UniRTM) | UniRTM (Uni Runtime and Tools Manager) — A cross-platform developer toolchain manager |

## Installation

### Add Tap

```sh
brew tap snowdreamtech/tap
```

### Install UniRTM

```sh
brew install --cask unirtm
```

Or in one step:

```sh
brew install --cask snowdreamtech/tap/unirtm
```

### Upgrade

```sh
brew upgrade --cask unirtm
```

### Uninstall

```sh
brew uninstall --cask unirtm
```

## Supported Platforms

| Platform | Architecture          |
| -------- | --------------------- |
| macOS    | Intel (x86_64)        |
| macOS    | Apple Silicon (arm64) |
| Linux    | Intel (x86_64)        |
| Linux    | ARM (arm64)           |

## About UniRTM

UniRTM is a cross-platform developer toolchain manager that unifies the management of various runtimes and development tools.

For more information, visit the [UniRTM project page](https://github.com/snowdreamtech/UniRTM).
